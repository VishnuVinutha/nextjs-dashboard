# Next.js Dashboard

A modern dashboard application built with Next.js, TypeScript, and Tailwind CSS.

## Features

- App Router
- Authentication
- Dashboard, Customers, Invoices pages
- Responsive UI

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```
2. Run the development server:
   ```
   npm run dev
   ```
